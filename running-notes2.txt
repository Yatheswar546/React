Snackbar
=> MUI Component 
=> Used to show notifications
=> <App /> Should be wrapped inside "<SnackbarProvider>"

=> we use "enqueueSnackbar" method to show notifications.
=> we get enqueueSnackbar from "useSnackbar()" hook

=> "notistack" is one such library with same functionalities and used widely in industries

Eg: import { useSnackbar } from "notistack";
    
    const MyButton = () => {
        
        const {enqueueSnackbar, closeSnackbar} = useSnackbar()
        
        return (
            <Button onClick = {() => enqueueSnackbar('Success')}>
                Show snackbar
            </Button>
        )
    }

=> Some commonly used props: 
    1) variant : for colors
    2) autoHideDuration: to close 
    3) anchorOrigin: for position (at a particular location)
    4) onClose: to close 
    5) style 
    6) transitionDuration 
    7) onEnter 
    8) onEntered
    9) onExit
    10) onExited
