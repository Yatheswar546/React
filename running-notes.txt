"useEffect" hook in React:

Problem Statement: infinite loop problem

import React, { useState } from "react";

export default function Products(){
    const [products, setProducts] = useState([]);

    function fetchData(){
        fetch("https://dummyjson.com/products")
            .then((res) => res.json())
            .then((data) => setProducts(data.products));
    }

    return <div>Products</div>
}

useEffect: used to handle side-effects in React application.
=> In the above example "fetchAPI" which is outside of React is making the website slow by infinite loop.
=> setTimeouts, LocalStorages, Read/Write files etc...

Syntax: useEffect((callback) => { Dependency Array})

Eg: without Dependency list
    
    useEffect(() => {
        console.log(products, "products");
        fetchData();
    });

Eg: with Dependency list

    useEffect(() => {
        console.log(products, "products");
        fetchData();
    }, []); //mounting

    In this fetchData() renders only once, because of Dependency List 
    This is also known as "Mounting Phase".

"key" attribute in lists:
=> each child in list should have unique "key" prop
=> because if we want to update/delete a particular item in that case it can be a problem as all cards are same.
=> So we need to have a unique "key" prop for each child.
=> By default the key is set to "Array Index".
=> If list elements can change order, use another unique key specific to each element
=> Array Index are not reliable because indices are not consistent.
=> Array Indices are used in case of non-CRUD operations.

Eg: using Array Index
return (
    <div>
        {products.map((product, index) => (
            <ProductCard data={product} key={index} />
        ))}
    </div>
);

import React, { useEffect } from "react";
import VideoCard from "./VideoCard";
import { Grid } from "@mui/material";
import axios from "axios";
import { enqueueSnackbar } from "notistack";

export default function Videos() {

    const [videos, setVideos] = useState([])

    async function fetchVideos(){
        try{
            let res = await axios.get("https://............");
            setVideos(res.data.videos);
            enqueueSnackbar("Fetched videos", {
                variant: "success", 
                preventDuplicate: true,
                anchorOrigin: {horizontal: "right", vertical: "top"}
            });
        }
        catch(err){
            console.log(err, "err");
            enqueueSnackbar(err.response.data.message, {variant: "error"});
        }
    }

    useEffect(() => {
        fetchVideos();
    }, []);

    return(
        <Grid container spacing={2} sx={{ justifyContent: "center" }}>
            {videos.map(video) => (
                <VideoCard data={video} key={video.id}/>
            )}
        </Grid>
    )    
}

Parallel API Calls in JS:

Promises:
=> It is an Object that represents eventual completion or failure of an asynchronous operation and its resulting value.
=> It returns state & value
=> It was used for calling API calls previously, at present we use fetch()

Problem: In case of multiple API Calls. It will increase the time as one API call will execute only after before API call is finished.

async function fetchData(){
    let startTime = new Date();
    let posts = await fetch("https://........./posts");
    let users = await fetch("https://........./users");
    let comments = await fetch("https://........./comments");
    let endTime = new Date();
    let timeTaken = endTime - startTime;
    console.log(timeTaken);
}

export { fetchData };

Solution: Promise Concurrency
            |-> Promise.all()
            |-> Promise.allSettled()
            |-> Promise.any()
            |-> Promise.race()

let consider timetaken for 3 tasks 
1 ==> 3 secs
2 ==> 5 secs
3 ==> 4 secs 

In normal execution : 3 + 5 + 4 = 12 secs

In Promise Concurrency:

after 3 secs 
1 ==> done 
2 ==> 2 secs 
3 ==> 1 sec 

after 4 secs 
1 ==> done 
2 ==> 2 secs 
3 ==> done

after 5 secs 
1 ==> done 
2 ==> done 
3 ==> done

let postApi = await fetch("https://........./posts");
let userApi = await fetch("https://........./users");
let commentsApi = await fetch("https://........./comments");
    

async function fetchData(){
    let startTime = new Date();
    
    let [postRes, userRes, commentRes] = await Promise.all([
        fetch(postApi),
        fetch(userApi),
        fetch(commentsApi),
    ]);

    let [posts, users, comments] = await Promise.all([
        postRes.json(), 
        userRes.json(), 
        commentRes.json(),
    ]);

    console.log(posts, users, comments, "res");

    let endTime = new Date();
    let timeTaken = endTime - startTime;
    console.log(timeTaken);
}

export { fetchData };

NOTE: Promise.all() accepts an array and returns an array.
